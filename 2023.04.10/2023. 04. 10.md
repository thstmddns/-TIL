### 2. 리스트 튜플

#### 2.1 리스트

+ 파이썬의 자료구조 중 하나

+ 여러 개의 값을 하나의 변수에서 관리하고 싶을 때 사용

+ 순서가 있는 수정가능한 객체의 집합

+ 대괄호([])로 작성되어지며, 리스트 내부의 값은 콤마로 구분

+ 값을 추가, 삭제 수정이 가능

+ 형태

  + 리스트명[요소1, 요소2, 요소3]

+ 값

  + 빈값
  + 숫자
  + 문자열
  + 숫자와 문자열 혼합
  + 리스트

+ 인덱싱

  + 무엇인가를 '가리킨다'는 의미 : 하나의 인덱스에 있는 값을 표시

+ 슬라이싱

  + 무엇인가를 ' 잘라낸다'는 의미 : 일정 범위의 인덱스에 있는 값을 표시
  + 리스트[start인덱스:end인덱스]
  + start 인덱스부터 end 인덱스 바로 전까지 값 변환

+ 리스트끼리의 연산

  + '+'

    + list4 = [1, 2, 3]
      list5 = [3, 4, 5, 6]
      list4 + list5

    + ```
      [1, 2, 3, 3, 4, 5, 6]
      ```

+ 리스트 값 추가

  + 리스트.append(값)
    + 리스트의 마지막 인덱스에 값이 추가된다.
  + 리스트.insert(인덱스, 값)
    + 원하는 인덱스에 값이 추가된다.

+ 리스트 값 수정

  + 해당하는 인덱스를 호출한 후 값을 선언하면 된다.

+ 리스트 값 삭제

  + del

    + 삭제하고자 하는 인덱스의 값을 삭제할 수 있다. 

    + 형태

      + list7 = [0, 1, 2, 3, 4, 5]
        del list7[1]
        list7

      + ```
        [0, 2, 3, 4, 5]
        ```

  + remove

    + 삭제하고자 하는 값을 삭제할 수 있다.
    + 형태
      + list7 = ['a', 'b', 'c', 'd', 'e']
        list7.remove('b')
        list7
      + ['a', 'c', 'd', 'e']

+ 리스트 정렬

  + sort()

    + 리스트에 있는 값을 오름차순으로 정렬

    + 형태 : 

      + list9 = [9, 77, 13, 51, 100, 3]
        list9.sort()
        list9

      + ```
        [3, 9, 13, 51, 77, 100]
        ```

  + reverse()

    + 리스트의 값을 바꾼다.

    + 형태: 

      + list9 = [9, 77, 13, 51, 100, 3]
        list9.sort()
        list9

      + ```
        [3, 9, 13, 51, 77, 100]
        ```

  + sort(reverse = True)

    + 리스트에 있는 값을 내림차순으로 정렬

    + 형태:

      + list10 = [9, 77, 13, 51, 100, 3]
        list10.sort(reverse = True)
        list10

      + ```
        [100, 77, 51, 13, 9, 3]
        ```

+ 리스트 인덱스 찾기

  + index()

    + 찾고자 하는 값의 위치 출력

    + 형태:

      + list11 = ['a', 'b', 'c', 'd', 'e', 'f']
        list11.index('c')

      + ```
        2
        ```

+ 리스트 값 삭제

  + pop()

    + 마지막 값을 출력 후 리스트에서 제거

      + 형태:

        + list12 = ['a', 'b', 'c', 'd', 'e', 'f']
          list12.pop()
          list12

        + ```
          ['a', 'b', 'c', 'd', 'e']
          ```

+ 리스트 값의 개수 확인

  + len()

    + 리스트의 값 개수 출력

      + 형태:

        + list13 = [0, 1, 2]
          len(list13)

        + ```
          3
          ```

    + 주의

      + 만약 
      + list14 = [ 0, 1,  2, [5]]라면 전체 길이는 4이다. 



#### 2. 2 튜플

+ 파이썬의 자료구조 중 하나
+ 여러 개의 값을 하나의 변수에서 관리하고 싶을 때 사용
+ 순서가 있는 집합
+ 소괄호 (())로 작성되어지며, 튜플의 내부의 내부 값은 콤마(,)로 구분
+ 추가, 수정, 삭제 불가능
+ 리스트에 비해 동작 속도가 빠름



#### 2.3 리스트와 튜플의 공통점과 차이점

+ 공통점
  + 타입과 상관없이 일련의 요소들을 가짐
  + 요소의 순서를 관리
+ 차이점
  + 리스트는 가변적이며 튜플은 불변적
  + 리스트는 요소의 개수가 명확하지 않고 유동적일 때 사용
  + 튜플은 요소의 개수가 명확하고 유동적이지 않을 때 사용



#### 3. 조건문에서 사용되는 in, not in

+ in: 찾고자 하는 값이 포함되어 있으면 True
+ not in : 찾고자 하는 값이 포함되어 있지 않으면  True

    + 조건식이 True일 경우 실행문장 반복
    + 형태 : 
      + while 조건식:
      + ​    실행문장
      + ​    실행문장
    + 주의할 점:
      +  while 문은 반복 횟수가 정해주지 않으므로 조건식이나 break문 등을 활용하여 무한 반복에 빠지지 않게 방지해야 한다.
  + for : 반복 횟수가 명확할 때
